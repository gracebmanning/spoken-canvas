Bracket syntax

[code] — continuous: re-runs every frame. Use for anything that should keep reacting to TIME.t / MIC.audio_level (creating a shape, an ongoing rotation, a growing size). Creator calls are safe to leave continuous — they reconcile onto the same object instead of duplicating it.
[!code] — one-shot: runs once, then is discarded. Use for a fixed-target tween/fade or a discrete action (clear(), remove(), ...). A fixed-target tween left continuous would re-assert its (stale) target forever and fight the next cue's tween.
[RESET()] — sugar for browser_2d.clear(); browser_3d.clear();
Any argument slot documented below as reactive accepts either a plain number (snap instantly, or tween over duration seconds if given) or a () => value function (evaluated every frame — read TIME/MIC inside it).

Ambient vars

TIME.t

Seconds since the script started (or last restart).

TIME.dt

Seconds since the previous frame.

MIC.audio_level

Smoothed microphone loudness, 0..1.

MIC.pitch

Smoothed fundamental frequency of the mic input, in Hz. -1 when silent or no clear pitch is detected — check for that before using the value.

SCRIPT.words_completed()

Count of spoken words passed so far (brackets excluded).

Helpers

approach(target, rate=0.1)

Eases toward a live, ever-changing value instead of snapping straight to it — drop it in anywhere a plain number is expected, e.g. move(c, x, approach(MIC.pitch)). Each call to approach() in a continuous (non-!) bracket remembers its own last value from frame to frame automatically — no variable to declare yourself. rate is how much of the remaining gap it closes per frame (closer to 0 = laggier/smoother, closer to 1 = snappier). Only works inside a continuous bracket — called from a one-shot [!...] (or anywhere with no per-tick call site to remember), it just returns target unchanged.

browser_2d (Paper.js)

Creators
circle(radius, color, x=960, y=540)

radius/color: number/string or reactive.

square(size, color, x=960, y=540)
rect(width, height, color, x=960, y=540)
ellipse(width, height, color, x=960, y=540)
arrow(fromX, fromY, toX, toY, color='black', thickness=3)
line(fromX, fromY, toX, toY, color='white', thickness=3)
text(content, x=960, y=540, size=24, color='white', align='center')

content may be a reactive () => value for live text.

box(width, height, color='white', x=960, y=540, borderWidth=2)

Outline only, no fill — good for diagrams.

Mutators
move(handle, x=960, y=540, duration=0)
rotate(handle, degrees=0, duration=0)

Absolute angle. Reactive degrees spins continuously.

scale(handle, sx=1.0, sy, duration=0)

Absolute factor, 1 = original size. sy defaults to sx.

fade(handle, value=1.0, duration=0)

Alias: opacity(...). 0 = invisible, 1 = opaque.

remove(handle)
clear()
